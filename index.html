Chicago
Information Design, Web Development

As part of Pratt's Web Mapping course, I created a website highlighting homicide in Chicago. I used Leaflet and d3.js to create information visualizations.

Pink Passenger Transportation
User Research and App Design

I did a user flow design for a transportation company app. After conducting field research by riding with potential users for hours, I organized their expectations for the app, sketching out different layouts for each screen, and then creating a mockup in Adobe Photoshop/Illustrator and animating user actions in Adobe AfterEffects.

This simulates a user checking the transportation route and then buying and using a ticket.

The Kullberg Foundation
Web Design and Development

This was an exciting venture into the world of web design and an opportunity to experience working with a client, creating a website from scratch, and designing user experience solutions for particular context.

My favorite aspect of the website is the audio solution. The client really wanted to have background music so I had to figure out how to integrate the music while still making the website user-friendly. This resulted in ditching the standard HTML5 player and creating a toggle function to control the music.
